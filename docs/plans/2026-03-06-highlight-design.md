# 형광펜 하이라이트 기능 설계

## 개요
책 챕터 페이지에서 텍스트를 드래그하여 형광펜(노란색)으로 마킹하고,
localStorage에 저장해 재방문 시 자동 복원하는 기능.

## 동작 방식
1. 챕터 본문에서 텍스트 드래그
2. 작은 "형광펜" 버튼 팝업 표시
3. 버튼 클릭 → 선택 텍스트에 노란 mark 스타일 적용
4. localStorage에 저장
5. 페이지 재방문 시 자동 복원
6. 하이라이트 위 마우스오버 시 ✕ 삭제 버튼 표시

## 저장 구조
- 키: `highlights_${bookSlug}_${chapterPath}`
- 값: `[{ id: string, text: string }]`

## 기술 방식
- ReactMarkdown 렌더링 후 useEffect로 .chapter-body DOM 순회
- 저장된 텍스트와 일치하는 텍스트 노드를 찾아 mark 태그로 래핑
- 중복 텍스트는 앞에서부터 순서대로 매칭

## 제약
- 동일 텍스트 중복 시 첫 번째만 하이라이트
- 책 내용 변경 시 기존 하이라이트 복원 불가

## 색상
- 노란색 단색 고정

## 저장소
- 브라우저 localStorage (기기별 저장, 로그인 불필요)
